设计说明

设计理念

图标采用画中画的视觉概念，通过大屏幕和小浮动窗口的组合来直观表达功能特性。

颜色方案

主色调渐变
#667EEA
#22C55E (激活)
#333 (屏幕)

设计元素

  • 主视频屏幕：代表原始视频播放区域
  • 播放按钮：表明这是视频相关功能
  • 小浮动窗口：代表画中画窗口
  • 连接线：显示两者之间的关系
  • 激活指示器：绿色圆点表示功能已激活
📋 使用说明
  • 16×16: 浏览器工具栏中的小图标
  • 32×32: 高分辨率显示器上的工具栏图标
  • 48×48: Chrome扩展管理页面显示
  • 96×96: Chrome Web Store和扩展详情页面
  • 128×128: Chrome Web Store和扩展详情页面
  • 激活状态: 当画中画功能激活时，图标会切换为绿色版本
