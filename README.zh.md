## 功能特点

-   **一键激活画中画**：通过点击工具栏图标或使用快捷键快速将视频切换到画中画模式
-   **多平台支持**：兼容各大主流视频网站
-   **黑白名单**：支持设置网站黑白名单，自定义哪些网站允许或禁止使用画中画功能
-   **多视频选择**：当页面中有多个视频时，可以选择要画中画的视频
-   **支持浏览器后台显示**：在浏览器关闭后，画中画窗口仍然可以显示，可继续续播放，直到用户手动关闭
-   **画中画快捷切换**：使用`←` `→`快捷键可以快速切换到下一个或上一个视频，方便用户在多个视频之间切换

## 快捷键

-   `Alt+P`：激活画中画模式
-   `ArrowLeft`：切换到上一个视频
-   `ArrowRight`：切换到下一个视频

## 浏览器兼容性
